# 点我! -轻飘飘的在线记事本

一个基于 Cloudflare Worker + D1 构建的轻量级在线记事本应用，支持用户注册/登录、笔记管理、分享功能，以及管理员后台管理。

## 功能特性
### 核心功能
- 笔记管理：创建、编辑、删除、查看笔记列表
-  笔记分享：生成公开分享链接（有效期30天）
-  用户系统：注册/登录/登出、修改密码
-  角色权限：区分普通用户（骑士）、高级用户（贵族）、管理员（皇帝）
-  自动清理：闲置无笔记的账号自动过期清理

### 角色权限说明
| 角色   | 核心权限                                  | 限制                          |
|--------|-------------------------------------------|-------------------------------|
| 骑士   | 基础笔记功能（创建/编辑/分享）| 最多创建8个笔记本，无笔记10天后过期 |
| 贵族   | 基础笔记功能（创建/编辑/分享）| 无笔记本数量限制，无笔记10天后过期 |
| 皇帝   | 所有功能 + 管理员后台                     | 永久有效，无任何限制          |


## 部署步骤
- 在*cloudflare*中创建*work*,名称任意
- 创建*D1 SQL数据库*，名称为*db*,创建*D1 SQL数据库*，名称为*db1*
- 将两个*D1 SQL数据库*绑定到*work*
- 绑定自定义域名即可访问


## 说明
*D1 SQL数据库*会自动初始化，无需配置

### 实例地址
https://clickhere.dpdns.org

### 技术栈
- 运行环境：Cloudflare Worker
- 数据库：Cloudflare D1（SQLite 兼容）
- 缓存 / 会话：Cloudflare KV
- 前端：原生 HTML/CSS/JS（无框架）
- 鉴权：Cookie + KV 会话存储
